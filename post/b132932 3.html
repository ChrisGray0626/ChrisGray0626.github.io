Hexo

Posted on 2021-09-25 Edited on 2022-07-22 In Tech , Hexo Views: Waline: Views: Symbols count in article: 3.6k Reading time ≈ 3 mins.

Hexo 是一个快速、简洁且高效的博客框架。Hexo 使用Markdown（或其他渲染引擎）解析文章，在几秒内，即可利用靓丽的主题生成静态网页。

常用命令

New

  • 新建文章
  • {layout}，可选参数
  • --path {path}，可选参数

Generate

  • 生成静态文件

Server

  • 启动本地服务器
  • 默认网址：http://localhost:4000/

Deploy

  • 部署网站

Clean

  • 清除缓存文件 (db.json)
  • 清除已生成的静态文件 (public)
