GitHub Pages

Posted on 2021-09-25 In Tech , GitHub Pages Views: Symbols count in article: 308 Reading time ≈ 1 mins.

GitHub Pages是免费的静态站点，三个特点：免费托管、自带主题、支持自制页面和Jekyll。

特点

  • 搭建简单
  • 完全免费
  • 静态脚本
  • 绑定域名
  • 专注内容

仓库创建

  • 使用GitHub用户名，每个用户下只能建立一个
  • 仓库命名要求：{username}.github.io
  • 仓库用以存放网页的静态资源

仓库设置

  • 点击Settings
  • 点击Pages
  • 配置域名
  • 配置分支
  • 选择主题

域名配置

  • 可自动解析域名
  • 域名管理系统中设置域名 // TODO
  • 仓库根目录创建文件，命名为CNAME，内容为域名
  • 开启HTTPS，勾选Enforce HTTPS

框架选择

  • 官方框架，Jekyll，基于Ruby
  • 第三方框架，Hexo，基于Node.js
